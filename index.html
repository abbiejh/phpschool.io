What is PHP School?

We are an interactive learning tool that will teach you the core skills of PHP through a set of workshops ranging from absolute beginner to advanced. Each workshop is run from the command line so if you are not familiar with this we have created a quick step by step tutorial to get you up to speed plus extra resources.

To Get Started

Before you can start our courses there are a few requirements which should be met:

  • You will need PHP with a version of at least 5.6 and a recent version of Composer installed
  • If you are running Windows, you will need a few more things which can be found on our our install page.
  • You will also need a Text Editor so you can work through the courses. You can use one of the following if you don't already have one: Atom, Sublime

For more help please visit our install page for more detailed instructions and troubleshooting.

And thats it! pick a workshop from below and follow the instructions!

Community workshops

At the core, PHP School is a workshop framework which allows you to run a set of exercises. It has been designed in a way that you can easily build your own workshops, focusing on any subject you please.

If you want your workshop featured here, drop us a line and we will get back to you.

Comprehensive documentation will be available soon which will detail how to build your own workshops. For now you can use Learn You PHP as a reference implementation. The package which provides the framework is PHP Workshop.

Need help?

If you need help or want to contribute to PHP School, you can chat on Slack, create issues and send PR's on GitHub. We also have a Twitter account where we will push out news and information!


If you have any issues with any of the workshops or just general getting started questions, you can create an issue here and someone will get back to you whenever it is possible. Feel free to also jump on and help other people with their issues. This type of project works best when everyone gets involved.
