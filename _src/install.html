Install

If you are having trouble installing PHP School then you may need to update your version of PHP before installing composer.

  • Linux - Debian Based
  • OSX
  • Windows

Linux

composer global require php-school/learn-you-php

learnyouphp

If /usr/local/bin is not writeable you will receive an error message and you will need to make sure Composer global bin directory is available in your $PATH environment variable. To learn more about the $PATH environment, click here.

If Composer's global bin directory is not available in your $PATH and the following to the end of your ~/.bashrc file:

export PATH=~/.composer/vendor/bin:$PATH

Requirements

PHP 5.6

If you have a PHP version less than 5.6, you will need to update

Note You can check your PHP version with php -v

sudo apt-get install software-properties-common

sudo add-apt-repository ppa:ondrej/php5-5.6

Composer

If you don't have composer installed

Note You can check if you have composer installed with composer -v

curl -sS https://getcomposer.org/installer | php

mv composer.phar /usr/local/bin/composer

chmod +x /usr/local/bin/composer

learnyouphp

Requirements

PHP 5.6

If you have a PHP version less than 5.6, you will need to update

Note You can check your PHP version with php -v

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

brew tap homebrew/dupes

brew tap homebrew/homebrew-php

brew install php56

export PATH="$(brew --prefix php56)/bin:$PATH"
