Install

If you are having trouble installing PHP School then you may need to update your version of PHP before installing composer.

  • Linux
  • OSX
  • Windows

Linux

Updating to PHP 5.6

If you have a version of PHP less than 5.6

Note: You can check your PHP version using php -v

sudo apt-get install software-properties-common

sudo add-apt-repository ppa:ondrej/php5-5.6

Installing Composer

If you need to install Composer

Note: You can check if you have composer installed with composer -v

curl -sS https://getcomposer.org/installer | php

mv composer.phar /usr/local/bin/composer

chmod +x /usr/local/bin/composer

OSX

Updating to PHP 5.6

If you have a version of PHP less than 5.6

Note: You can check your PHP version using php -v

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

brew tap homebrew/dupes

brew tap homebrew/homebrew-php

brew install php56

export PATH="$(brew --prefix php56)/bin:$PATH"

Installing Composer

If you need to install Composer

Note: You can check if you have composer installed with composer -v
